¿Qué es?
El Libro de Guardia es una aplicación web offline diseñada para médicos residentes. Contiene módulos clínicos de manejo rápido, calculadoras y referencia de fármacos para usar directamente en la guardia, sin conexión a internet.
🔗 App disponible en: lautarodifonzofasta.github.io/ellibrodeguardia/
---
Contenido
Categoría	Módulos
🚨 Emergencias críticas	PCR, Shock, Sepsis, Anafilaxia
🫀 Cardiovascular	Dolor torácico, Arritmias, IC aguda, Crisis HTA, TEP, TVP
🫁 Respiratorio	Disnea, Neumonía, EPOC, Asma GINA 2026
🧠 Neurológico	ACV, Convulsiones, Cefalea, Delirium
🤰 Ginecobstétrico	Emergencias obstétricas, HTA gestacional, EPI
🧬 Psiquiátrico / Toxicología	Agitación, Intoxicaciones, Toxidromes
🧮 Calculadoras	GCS, SOFA, Wells, CURB-65, HEART, NIHSS y más
💊 Referencia	Drogas de emergencia con dosis
64 módulos · 11 calculadoras · 100+ fármacos con dosis
---
Características
✅ 100% offline — funciona sin internet una vez instalada
✅ Instalable en iOS (Safari → Agregar a inicio) y Android (Chrome → Instalar app)
✅ Basada en guías actualizadas — GINA 2026, ACOG, FIGO 2022, CDC 2024, SADI 2025
✅ Un solo archivo HTML — sin dependencias externas
✅ Diseñada para guardia — navegación rápida, tipografía legible, modo oscuro
